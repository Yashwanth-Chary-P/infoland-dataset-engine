# Dataset Statistics

Generated statistics are written by `python scripts/run_preprocessing.py`.

Primary outputs:

- `data/reports/inspection/`
- `data/reports/validation/`
- `data/reports/statistics/overall_stats.json`
- `data/reports/statistics/master_dataset_stats.json`

The generated `data/master/master_properties.json` file is the future source of truth for downstream synthetic datasets.
